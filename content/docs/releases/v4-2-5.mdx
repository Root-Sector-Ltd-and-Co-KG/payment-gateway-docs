## [4.2.5] - 2026-04-07

### Changed Components

- **admin-backend** v4.1.7
- **admin-frontend** v4.1.6
- **main-backend** v4.1.5

### Added
- **admin-backend**: Implement tax rounding method configuration and validation in invoice calculations.
- **admin-backend**: Implement failing field service for encryption tests and update test cases.
- **admin-frontend**: Add tax preview functionality to transaction items and summary components.

### Changed
- **main-backend**: Update tax calculation methods to support configurable rounding and defer tax snapshot for pending transactions.

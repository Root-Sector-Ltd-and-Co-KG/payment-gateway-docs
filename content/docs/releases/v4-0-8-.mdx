# Release 4.0.8

- Channel: stable
- Published: 2026-03-17T21:00:22Z
- Release tag: v4.0.8

## [4.0.8] - 2026-03-17

### Changed Components

- **admin-backend** v4.0.7
- **admin-frontend** v4.0.6
- **main-backend** v4.0.4

### Added
- **admin-backend:** Streamlined user role assignment and enhanced registration flow.
- **main-backend:** Introduced licensing endpoint URL configuration and enhanced license service behavior.

### Changed
- **admin-frontend:** Improved version check logic and enhanced license error guidance messages.
- **main-backend:** Updated dependencies including phonenumbers to v1.6.12, bytedance/gopkg to v0.1.4, and latest versions of googleapis.

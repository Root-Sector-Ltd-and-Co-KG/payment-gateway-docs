# Release 4.4.3

- Channel: stable
- Deployment target: production
- Published: 2026-06-10T19:24:02Z
- Release tag: v4.4.3

## [4.4.3] - 2026-06-09

### Admin Panel
- Introduced a new user interface dialog for managing and canceling recurring payment schedules.
- Improved accessibility across administrative dashboard components for better screen reader and keyboard navigation.
- Standardized the default currency display to EUR across transaction modals and formatting utilities.
- Refactored tax management to use a streamlined tax evidence collection system for improved compliance tracking.

### APIs And Integrations
- Enhanced e-invoice handling and data processing logic for improved compatibility with external billing standards.
- Integrated an updated tax evidence library to streamline how tax data is validated and stored during transactions.

### Checkout And Customer Portal
- Implemented a new PDF generation flow for customer documents and receipts, including updated localization support.
- Added validation for payment URLs to ensure secure and verified redirection during the checkout process.
- Updated currency formatting defaults to improve consistency for end-users across different locales.

### Deployment And Operations
- Updated the reverse proxy configuration to utilize Caddy v2 storage module paths.
- Optimized Docker build contexts for more efficient image deployment and reduced build times.

### Security
- Centralized security header management within the reverse proxy to ensure consistent protection across all endpoints.
- Enhanced client IP header handling to improve the accuracy of security logging and threat detection.
- Updated core payment libraries and internal communication headers to strengthen the system's security posture.
- Improved security configurations for document and PDF handling to protect sensitive generated files.

### Fixes
- Resolved an issue where transaction modals and formatting utilities did not default to the correct currency.
- Fixed inconsistencies in the tax evidence collection process during the payment lifecycle.

# Release 4.4.4

- Channel: stable
- Deployment target: production
- Published: 2026-06-12T17:24:00Z
- Release tag: v4.4.4

## [4.4.4] - 2026-06-10

### Admin Panel
- Introduced standardized canonical item types across the management interface to ensure more consistent product categorization and data integrity.

### APIs And Integrations
- Improved API item type handling so custom and unknown item categories are preserved consistently in invoice, tax, and integration payloads.

### Fixes
- Resolved an issue where unknown and custom item types were incorrectly mapped during backend processing, ensuring accurate data handling for non-standard inventory.

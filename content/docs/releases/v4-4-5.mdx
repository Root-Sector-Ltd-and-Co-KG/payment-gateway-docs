# Release 4.4.5

- Channel: stable
- Deployment target: production
- Published: 2026-06-13T15:56:17Z
- Release tag: v4.4.5

### Admin Panel
- Added support for authenticated access to release metadata using secure IDs and secrets, allowing for controlled retrieval of system versioning information.
- Implemented a backfill process to standardize and canonicalize legacy item types, ensuring data consistency across administrative records.

### Checkout And Customer Portal
- Introduced client-side rate limiting to enhance platform stability and protect the interface from high-frequency request bursts.

### Deployment And Operations
- Configured the reverse proxy to include the `X-Forwarded-For` header, enabling operators to accurately track and log original client IP addresses.

### Security
- Enhanced request security by implementing frontend rate limiting to mitigate automated abuse.
- Improved audit logging accuracy by ensuring the preservation of client IP addresses through network proxy layers.

# Release 4.2.0

- Channel: stable
- Deployment target: production
- Published: 2026-03-31T20:11:33Z
- Release tag: v4.2.0

## [4.2.0] - 2026-03-31

## Customer-Facing Changes

### Admin Panel

- Added IPN retry controls and webhook delivery details on transaction detail pages.
- Added Stripe webhook API version guidance and enhanced setup instructions.
- Added organization member management and user settings configuration.
- Added tax rate, rule, and settings management.
- Improved address handling, validation behavior, and payment link settings descriptions.
- Improved checkout detail and creation screens for better user flow.
- Updated provider and transaction screens to clarify wire transfer terminology and instructions.
- Improved recurring invoice handling and calculation behavior.

### APIs And Integrations

- Added a new retry IPN endpoint and enhanced transaction handling capabilities.
- Added organization user management.
- Added ACME HTTP challenge management and domain activation retry behavior.
- Added support for creating manual credit notes for invoices.
- Added a `PaymentURL` field to invoice and session response types.
- Added VAT validation and tax preview.
- Improved checkout session handling, item mapping, and email formatting.
- Refined payment confirmation descriptions and tax decision handling.
- Improved portal session management and transaction service methods.

### Checkout And Customer Portal

- Expanded checkout payment handling and localized finalization messages.
- Added an optional company name field and improved VAT ID validation messages during checkout billing entry.
- Added VAT ID validation and tax preview features on the checkout payment form.
- Updated localization guidance to clarify currency selection for wire and Wise payments.

### Deployment And Operations

- Added webhook processing status updates for Stripe and GoCardless.
- Clarified VAT ID validation behavior in the API documentation.
- Updated reverse proxy routing to support ACME HTTP validation tokens and mandatory HTTPS redirection.

## Deployment Notes

Use the updater from `payment-gateway-deploy` to install this release. Exact image tags are resolved from release metadata and the deployment tooling instead of being duplicated in these notes.

## [4.2.7] - 2026-04-19

## Customer-Facing Changes

### Admin Panel

- Improved transaction filtering with currency support and clearer request handling.
- Added license requirement checks for license-gated pages.
- Improved dashboard transaction previews and data loading.

### APIs And Integrations

- Added currency filter handling in transaction retrieval.
- Added retention days setting for geolocation during organization creation.
- Improved provider request and response diagnostics.
- Improved invoice generation reliability.

### Deployment And Operations

- Added retention days configuration and update handling in geolocation settings.
- Improved remote licensing status handling and caching.

# Release 4.3.0

- Channel: stable
- Deployment target: production
- Published: 2026-04-25T17:09:31Z
- Release tag: v4.3.0

## [4.3.0] - 2026-04-25

## Customer-Facing Changes

### Admin Panel

- Added country name resolution for improved display in charts and reports.
- Improved VAT OSS reporting with chargeback and refund details.

### APIs And Integrations

- Improved transaction handling with VAT OSS and chargeback support.
- Improved error responses for transaction handling and added configuration validation for PayPal providers.
- Improved API response structure with warnings for degraded states.
- Added PayPal provider integration.
- Improved transaction handling reliability.

### Checkout And Customer Portal

- Added PayPal payment integration with support for advanced card fields and error handling.
- Improved checkout service asset handling, rate limiting, and graceful shutdown.
- Improved checkout experience with localization updates.

### Fixes

- Refined Content Security Policy to enhance XSS protection while allowing payment SDKs.
